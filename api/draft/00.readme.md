# NEWEGGCENTRAL REST API目标
NeweggCentral REST API允许Newegg内部和外部开发人员通过编程方式访问Newegg系统的各种对象与资源。
NeweggCentral REST API需最大限度地满足平台无关性。

如何设计一套合理、好用、规范的API。

在本文中，REST API， Web API, REstful API, Restful Services都是指同一个意思。
