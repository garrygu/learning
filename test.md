## Dockerfile Commands (Instructions)
- **FROM**  *Set base image*  
    - Usage: FROM '<image>'
    -   Example:  
            ```
            FROM ubuntu:latest  
            ```
    -       FROM ubuntu:latest
    - A valid Dockerfile must have **FROM** as its first non-comment instruction
    - **FROM** can appears multiple times within a single Dockerfile in order to create multiple images
    -
        > FROM ubuntu:latest
